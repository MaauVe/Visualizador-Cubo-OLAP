Bienvenido al Dashboard de Análisis OLAP

Esta aplicación demuestra las operaciones de un Cubo OLAP (Procesamiento Analítico en Línea) utilizando Flask y Pandas.

Use la barra de navegación de arriba para explorar las diferentes vistas:

  • Cubo Completo: Muestra la vista pivoteada principal (Producto/Región vs Año/Trimestre).
  • Cara: Muestra una vista 2D diferente (Pivot) del cubo (Año vs Región).
  • Sección: Permite filtrar (Dice) el cubo por múltiples dimensiones (Años, Regiones) y ver el resultado agregado.
  • Drill-Through: Permite "perforar" una celda específica del cubo para ver los datos crudos que la componen.
